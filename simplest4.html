步驟簡單說明

1.打開Ana3_lite2後按start

2.到cmd畫面cd tmp

3.git clone https://github.com/2015fallhw/2015fallcadpb.git

4.cd 2015fallcadpb

5.git checkout gh-pages

6.回倒Y槽按2015fallcadpb資料夾

7.進去後按user資料夾裡面複製9999資料夾貼上後重新命名改為自己的學號

8.進到自己學號的資料夾後再點進去Pelican裡面的content資料夾，將所有檔案刪除

9.使用LEO Editor 打開班上倉儲的reveal檔

10.copy @path 9999後 paste node 重新命名為@path 自己的學號

11.進入剛剛修改的node 刪除檔名有9999的檔案

12.到.py的檔案內修改9999為自己的學號

13.開啟個人倉儲的reveal檔，分別複製投影片及各周的MD檔

14.再分別貼至班上倉儲的檔案對應位置

15.確認無誤後save 按一下自己學號的資料夾後再按 user pelican

16.回到cmd的畫面 git add .

17.git commit -m "40323203"

18.git pull origin gh-pages

19.git push origin gh-pages

20.最後自己的網頁網址為http://2015fallhw.github.io/2015fallcadpb/user/自己的學號/

自評與心得

自評分數:70

這一次是學習如何把自己的倉儲推回去班上，步驟上覺得有點麻煩，因為在推回去的過程中還須考慮班上是否還有人跟你一起在操作，所以就必須先git pull確認 ，假如還有人在操作就會顯示對方的學號一在上面，另外的問題就是班上有些人在git push之前沒有先git pull，這樣就會導致某些人在Leo裡的檔案會不見，雖然在連結自己學號的網頁是顯示，可是在還沒繳交作業之前就會怕自己在Leo裡面的檔案不見，所以我覺得班上只要有人會，那就麻煩他們教學，這樣就不會導致有人的檔案被覆蓋，當然也要有人去主動問，所以這次我花了很久的時間在研究作業，雖然還是覺得自己只是照著步驟操作，但是至少我知道只要有一個步驟漏掉，這樣班上的Leo就會打不開而爆炸，所以應該還是算有學習到一些東西。
