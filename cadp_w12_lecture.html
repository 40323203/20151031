第 12 週開始的規劃

2015Fall 電腦輔助設計實習

操作步驟說明1

  1. 至http://wordpress-2015course.rhcloud.com/
  2. 選擇2015Fall課程裡的copy.com: https://copy.com/zpYyuIcuB3fHAdr0 (fix.zip)
  3. 回到隨身碟點Ana3_lite2→data→SciTE→SciTE.exe
  4. 開啟後將下載的fix資料夾裡的fix.bat和start.bat用SciTE.exe打開
  5. 回到網站輸入leo-editor github→點進去後按releases→下載broke-abbrev

操作步驟說明2

  1. 下載後將此資料夾複製到fix資料夾貼上
  2. 回到SciTE.exe
  3. 將fix.bat裡面多一行copy leo-editor-broke-abbrev.zip C:\Ana3_2015\data\→儲存
  4. 將start.bat裡面第35行改為pip install leo-editor-broke-abbrev.zip→儲存
  5. 全部修改完回去點start後leo就會是5.1版本了
