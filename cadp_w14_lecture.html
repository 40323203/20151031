操作步驟說明1

  1. 先把CREO軟體打開→open零件圖和組合圖
  2. 按檔案另存新檔選為stl檔
  3. 之後會跳出一個畫面→選ASCII→輸入0/0→確定
  4. 回到網頁至個人github新創倉儲為group3
  5. 到cmd git clone 網址→再到Y槽將group3資料夾打開

操作步驟說明2

  1. 複製貼上stl檔到資料夾→推回去遠端
  2. 回到github網站→group3→按上傳的stl檔
  3. 按右鍵顯視原始碼→複製網址
  4. 放置班上LEO→推回去遠端→完成

自評與心得

自評分數:65

第14週的課程是教學如何將STL檔推回去遠端並且用3D的方式呈現，剛開始要先將畫好的CREOL檔轉存為STL檔，並且新創小組倉儲，完成後主要是可以將零件圖用3D的方式呈現在網站上，一方面可以將圖翻旋360度觀看，也可以將組合圖呈現更完整，所以這一週讓我有所大開眼原來可以這樣呈現3D 圖。
